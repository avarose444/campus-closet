# HOW TO RUN
To run Campus Closet:

First (in campus-closet directory), run _npm install_ 

Then, run _npm start _


Finally, navigate to the local site, and sign into the login page with emilysher


# TOOLS USED
React: JavaScript UI library for building the frontend. 

Vite: development server used.

ESLint: Linting configuration (eslint.config.js) to enforce code quality/style. 

Node.js / npm: Project likely uses npm (package.json, package-lock.json) for dependency management. 

Firebase: backend for user authorization and data storage.

HTML/CSS/JavaScript: Standard web languages used.
